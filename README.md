# Hello Braintree

Original post: https://www.dennisokeeffe.com/blog/braintree-node-react

## Quickstart

```bash
git clone https://github.com/okeeffed/hello-braintree.git

# For the server
cd hello-braintree/server
yarn install
node server.js

# For clientside
cd hello-braintree/clientside
yarn install
yarn start
```
